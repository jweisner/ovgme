Création de snapshot

Pour créer un nouveau snapshot pour le jeux actuellement sélectionné, allez dans le menu "Game", sélectionnez "Snapshot" puis "New...". Une nouvelle fenêtre vous invitera à commencer le processus de création de snapshot. La création du snapshot peut prendre un long moment étant donné qu'il doit lire tous les fichiers du jeu pour créer les checksum xxHash, vous pouvez annuler le processus de création à tout moment.
